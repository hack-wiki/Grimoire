# About

Grimoire is a lightweight static site generator designed for creating beautiful documentation websites from Markdown files.

## Features

- **Simple** - Just write Markdown, run one command
- **Fast** - Pure Python, no heavy dependencies
- **Flexible** - Customizable templates and styling
- **Modern** - Dark/light themes, mobile responsive
- **SEO Ready** - Automatic sitemap, meta tags, structured data

## Philosophy

Documentation should be easy to write and beautiful to read. Grimoire focuses on getting out of your way so you can focus on content.

## Open Source

Grimoire is MIT licensed and open source. Contributions are welcome!
